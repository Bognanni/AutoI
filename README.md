# AutoI - Tesi Triennale

Questo repository contiene il codice sorgente del mio progetto di Tesi Triennale. 
Il sistema è un prototipo RPA in grado di eseguire azioni automatiche, come il movimento
del mouse e lo scroll, al fine di concludere la configurazione iniziale di software.

## Struttura del progetto
- `Main.py` / `GenericMainByImage.py`: Script principali di esecuzione.
- `Actions/`, `Movement/`, `ByName/`: Moduli per la gestione della logica.
- `Utilities/`: Script di supporto.

## Tecnologie usate
- Python 3.12
- pyautogui
- Movement
